## Overview

The LK (Little Kernel) payload hooks into the Android bootloader to control boot behavior, unlock bootloader restrictions, and enable fastboot access. It runs early in the boot chain and can modify boot image loading behavior.

**Source:** `lk-payload/main.c:81`

## GPT Partition Detection

The payload parses the GPT to locate key partitions:

```c
static void parse_gpt() {
    uint8_t raw[0x800] = { 0 };
    struct device_t *dev = get_device();
    dev->read(dev, 0x400, raw, sizeof(raw), USER_PART);
    
    for (size_t i = 0; i < sizeof(raw) / 0x80; ++i) {
        uint8_t *ptr = &raw[i * 0x80];
        uint8_t *name = ptr + 0x38;
        uint32_t start;
        memcpy(&start, ptr + 0x20, 4);
        
        if (memcmp(name, "b\x00o\x00o\x00t\x00\x00\x00", 10) == 0) {
            g_boot = start;
        } else if (memcmp(name, "r\x00\x65\x00\x63\x00o\x00v\x00\x65\x00r\x00y\x00\x00\x00", 18) == 0) {
            g_recovery = start;
        } else if (memcmp(name, "l\x00k\x00\x00\x00", 6) == 0) {
            g_lk = start;
        } else if (memcmp(name, "M\x00I\x00S\x00\x43\x00\x00\x00", 10) == 0) {
            g_misc = start;
        }
    }
}
```

**Implementation:** `lk-payload/main.c:56-79`

<Note>
The GPT parser reads partition table entries from offset 0x400 (block 1) in the user partition. Each entry is 128 bytes (0x80) and contains the partition name in UTF-16LE format.
</Note>

## Boot Mode Control Flags

### MISC Partition Flags

The payload reads control flags from the MISC partition:

| Flag | Offset | Action |
|------|--------|--------|
| `boot-amonet` | 0x00 | Boot to fastboot (temporary) |
| `FASTBOOT_PLEASE` | 0x00 | Boot to fastboot (persistent) |
| `boot-recovery` | 0x00 | Boot to recovery mode |
| `boot-system` | 0x00 | Force normal system boot |
| `UART_PLEASE` | 0x10 | Enable UART output |

**Implementation:** `lk-payload/main.c:141-190`

### Flag Processing Example


<Note>

## Boot Image Hooking

The payload hooks the `read` function to modify boot images as they're loaded:

```c
size_t read_func(struct device_t *dev, uint64_t block_off, 
                 void *dst, uint32_t sz, uint32_t part) {
    if (block_off == g_boot * 0x200 || block_off == g_recovery * 0x200) {
        printf("demangle boot image\n");
        
        // Check if Y-cable is connected and should redirect to recovery
        if (block_off == g_boot * 0x200 && *y_cable == 1 && 
            !boot_system && *boot_reason == 0) {
            printf("Boot into recovery to display bootmenu\n");
            block_off = g_recovery * 0x200;
        }
        
        // Skip first 1KB of boot image (exploit payload area)
        if (sz < 0x400) {
            ret = original_read(dev, block_off + 0x400, dst, sz, part);
        } else {
            void *second_copy = (char*)dst + 0x400;
            ret = original_read(dev, block_off, dst, sz, part);
            memcpy(dst, second_copy, 0x400);
            memset(second_copy, 0, 0x400);
        }
    } else {
        ret = original_read(dev, block_off, dst, sz, part);
    }
    return ret;
}
```

**Implementation:** `lk-payload/main.c:32-54`

<Note>
The boot image hook removes the first 1KB from boot/recovery images. This area is used to store the exploit payload and must be removed before the bootloader processes the Android boot image header.
</Note>

## Security Bypass

### Unlock Status Bypass

Forces the bootloader to report as unlocked:

```c
// device_is_unlocked()
patch = (void*)0x41E01D34;
*patch++ = 0x2001; // movs r0, #1
*patch = 0x4770;   // bx lr
```

**Implementation:** `lk-payload/main.c:211-214`

### Command Verification Bypass

Disables Amazon's fastboot command verification:

```c
// amzn_verify_unlock() - enable all commands
patch = (void*)0x41E01EA0;
*patch++ = 0x2000; // movs r0, #0
*patch = 0x4770;   // bx lr
```

**Implementation:** `lk-payload/main.c:216-219`

<Note>
These patches modify ARM Thumb-2 instructions to force functions to return specific values. `movs r0, #1` sets the return value to 1 (true), while `bx lr` returns immediately.
</Note>

## Boot Flow

### Standard Boot Sequence

1. Microloader loads LK payload from eMMC
2. LK payload restores overwritten memory
3. Parse GPT to locate partitions
4. Check boot mode flags (MISC, microloader)
5. Apply security patches
6. Install boot image hook
7. Jump to original LK entry point
8. LK loads boot/recovery image (via hook)
9. Boot continues normally

**Implementation:** `lk-payload/main.c:81-237`

### Fastboot Boot Sequence

1. Steps 1-5 same as above
2. Set `*g_boot_mode = 99` (fastboot)
3. Apply patches and install hooks
4. Jump to LK
5. LK enters fastboot mode with unlocked commands

### Recovery Boot Menu

When Y-cable is detected:

```c
if (block_off == g_boot * 0x200 && *y_cable == 1 && 
    !boot_system && *boot_reason == 0) {
```

## Usage Example

### Setting Boot Flags

```python
import struct

class LKControl:
    def __init__(self, stage2):
        self.s2 = stage2
        # Find MISC partition by reading GPT
        self.misc_block = self.find_partition('MISC')
    
    def set_fastboot_flag(self, persistent=False):
        """Set flag to boot to fastboot mode"""
        flag = b'FASTBOOT_PLEASE' if persistent else b'boot-amonet'
        msg = flag.ljust(32, b'\x00')
        
        self.s2.switch_partition(8)  # USER_PART
        self.s2.write_block(self.misc_block, msg[:512])
    
    def set_recovery_flag(self):
        """Set flag to boot to recovery"""
        msg = b'boot-recovery'.ljust(512, b'\x00')
        self.s2.switch_partition(8)
        self.s2.write_block(self.misc_block, msg)
    
    def enable_uart(self):
        """Enable kernel UART output"""
        # Read existing message
        msg = bytearray(self.s2.read_block(self.misc_block))
        # Set UART flag at offset 0x10
        msg[0x10:0x10+11] = b'UART_PLEASE'
        self.s2.write_block(self.misc_block, bytes(msg))
    
    def clear_flags(self):
        """Clear all boot flags"""
        msg = b'\x00' * 512
        self.s2.switch_partition(8)
        self.s2.write_block(self.misc_block, msg)

# Usage
lk = LKControl(stage2)

# Boot to fastboot mode
lk.set_fastboot_flag(persistent=True)
lk.enable_uart()
stage2.reboot()

# Device will boot to fastboot with unlocked commands
```
